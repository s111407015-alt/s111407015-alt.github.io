網站設計與賞析
課程成果作品展示

關於這堂課程:

網頁（web page）是一個適用於全球資訊網和網頁瀏覽器的檔案，而網站（web site）則是網頁的集合，它存放在世界某個角落的某一部或一組電腦中，而這部電腦必須是與網際網路相連。現今，網站已是我們生活中獲取資訊的主要來源，如何評斷一個網頁設計的優劣？如何設計出一個具有質感且使用者友善的網頁？已是應具備的資訊素養。本課程由淺入深，教師透過實例的講解與示範，除原理原則的說明外，更透過帶領學生做中學、學中做，逐步建構出網站。並提供說明與講解的教學影片供學生課前預習與課後複習，同步搭配課堂的練習，讓學生能聽得懂、學得會與做得到。
